# Minerva School — Prototype (Light Climate Palette)

A lightweight, **interactive prototype** aligned to your deck, restyled with a climate‑inspired palette (teal/green/blue on white). It adds **Join / Log in**, **onboarding**, a **student dashboard** with the six‑step journey, adaptive **Lesson** (branching), **Reflection**, **Peer Matching** with **Teacher Override**, **Project Hub**, and **About/Ethics**.

## Files
- `index.html` · `styles.css` · `app.js` · `assets/`

## Use your real logo
Replace `assets/logo.png` with your uploaded Minerva logo (same filename) before publishing.

## 1‑minute deploy (GitHub Pages)
1. Create a repo (e.g., `minerva-climate-prototype`).
2. Upload the **contents** of this folder (not the ZIP): HTML/CSS/JS + `assets/`.
3. Settings → Pages → **Deploy from a branch** → `main` + **/** (root) → Save.
4. Your URL: `https://<your-username>.github.io/minerva-climate-prototype/`.

## Demo script
1) **Join** → Create account → **Onboarding** (AI assesses).  
2) **Dashboard** → see the **six‑step checklist**.  
3) **Lesson** → branch (hint/wrong → scaffold; correct → challenge).  
4) **Reflection** → confidence.  
5) **Peers** → toggle **Teacher Override**.  
6) **Projects** → City Model + Vocabulary.  
7) **About** → ethics & mastery model.

## Notes
- Front‑end only, no storage; safe for demo.  
- Colors are chosen for contrast/accessibility while matching a climate‑action vibe.  
- You can tweak color variables in `styles.css`.
