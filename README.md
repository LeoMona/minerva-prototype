# Minerva School – AI Global Learning Prototype

A prototype for a global AI-powered school platform, designed for K-12 learners.

## Features
- International peer community
- Dynamic lessons and projects
- Wellness chatbot for mood tracking
- Responsive design for web

## Tech Stack
- HTML, CSS, JavaScript
- Static GitHub Pages deployment

## How to Run
1. Clone or download the repository
2. Open `index.html` in a browser
3. Click the chatbot button to test wellness check

## Folder Structure
- `assets/` – images and logo
- `css/` – styles
- `js/` – scripts (app logic, data, chatbot)
