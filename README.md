# Minerva School — Light Prototype (GitHub‑ready)

A lightweight, **light‑themed** concept prototype aligned to your Canva presentation (pastel teal/violet, clean white cards). It demonstrates the Minerva learner flow and ethics guardrails.

## What’s inside
- Home → Lesson (branching) → Reflection → Peer Matching (teacher override) → Project Hub → Dashboard → About/Ethics
- Light color palette matching the presentation
- No backend, no persistence, **no personal data**

## 1‑minute deploy (GitHub Pages)
1. Create a new repo (e.g., `minerva-light-prototype`).
2. Upload these files (drag‑and‑drop): `index.html`, `styles.css`, `app.js`, `README.md`, `assets/`.
3. Settings → Pages → **Deploy from a branch** → `main` → **/** (root) → Save.
4. Your URL: `https://<your-username>.github.io/minerva-light-prototype/`

## Replace the logo
- Put your logo file as `assets/logo.svg` (or `logo.png`) and update `index.html` if needed.

## Customize
- Edit the pastel colors in `styles.css` (variables at the top).
- Add more lesson items and branching in `app.js`.

This prototype is for demonstration and coursework purposes.
